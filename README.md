# banking-management-system
Console-based Bank Management System developed using Core Java with account management, deposit, withdrawal, balance tracking, and interest calculation features.

## Features
- Create customer bank account
- Deposit amount
- Withdraw amount
- Check account balance
- Calculate interest
- Display customer details
- Date and time tracking for transactions

## Technologies Used
- Java
- OOP Concepts
- Collections
- Scanner Class
- LocalDate & LocalTime

## Concepts Covered
- Classes & Objects
- Encapsulation
- Inheritance
- Method Overloading
- Exception Handling
- Menu-driven programming
